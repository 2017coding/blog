# 浏览器页面渲染过程
## 前言

 浏览器内核是支持浏览器运行的核心程序，分为两部分：渲染引擎和JS引擎。

 渲染引擎在不同浏览器上存在区别，目前市面上常见的浏览器内核分为：
- Trident(IE)
- Gecko(火狐)
- Blink(Chrome、Opera)
- Webkit(Safari)
    
Webkit是当下浏览器世界的霸主，主要以Webkit为例子，做现代浏览器渲染过程分析。

## 页面加载过程

先大致整理一下访问页面到页面展示出来的过程：
- 浏览器将请求的URL交给DNS服务器解析，得到域名的IP地址
- 向这个IP地址的机器发生HTTP(s)请求
- 服务器收到、处理并返回HTTP请求
- 浏览器得到返回内容渲染成页面
